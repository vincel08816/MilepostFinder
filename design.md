# Milepost Website JavaScript Project (no backend)

1. use location services to determine the user's lat long coordinate
2. use the lat long coordinate to find the nearest milepost
3. display the nearest milepost on a leaflet map (zoom in)
4. show the nearest the two nearest mileposts on the map
5. find the direction of the nearest road segment to the user (Northbound, Southbound, Eastbound, Westbound)
6. Do not send data outside of the user's browser
